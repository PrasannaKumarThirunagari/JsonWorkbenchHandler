# Extensions folder

Browser extensions use the **same files as the web app** in the **repository root** (`index.html`, `app.js`, `styles.css`, `theme-init.js`, `manifest.json`, etc.).

Load **unpacked** / **temporary add-on** from the project root directory (the folder that contains `manifest.json`), not from this `extensions/` path.

See the **Browser extension** section in the root `README.md`.
